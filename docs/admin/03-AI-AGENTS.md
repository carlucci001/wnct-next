# AI Agents

## Overview
The platform uses AI agents to automate tasks. Configure them in **Settings > AI Agents**.

## Available Agents
1. **Journalist Agent**: Writes articles based on prompts or data sources.
2. **Editor Agent**: Reviews content for grammar, style, and tone.
3. **SEO Agent**: Optimizes headlines and meta descriptions.
4. **Social Agent**: Generates social media posts for articles.
5. **GEO Agent**: Tag articles with location data.
6. **Fact-Checking Agent**: Verifies claims against trusted sources.

## Article Creation
To use the Journalist Agent:
1. Go to **Articles > New AI Article**.
2. Enter a topic or paste a source URL.
3. Select the agent persona (e.g., "Investigative Reporter").
4. Click "Generate".

## Image Generation
Use the **Image Gen** tool in the Media Library or Article Editor to create custom images using DALL-E or similar models.
