# Plugins & APIs

## API Configuration
Configure external services in **Settings > APIs**.

### OpenAI
- **API Key**: Required for most AI features (GPT-4).
- **Model Selection**: Choose default models.

### Gemini
- **API Key**: Required for specific analysis tasks and this Chat Assistant.

### Perplexity
- **API Key**: Used for real-time web research and fact-checking.

### ElevenLabs
- **API Key**: Enables text-to-speech features for audio articles.

### Weather
- **Provider**: Open-Meteo (default) or others.
- **Location**: Set the default region for weather widgets.

### Stripe
- **Keys**: Publishable and Secret keys for subscription payments.
- **Webhooks**: Configure endpoints for payment events.
