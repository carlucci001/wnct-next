# User Management

## Users
Manage users in **Users**.
- **List**: View all registered users.
- **Search**: Find users by email or name.
- **Edit**: Change roles or update profile info.

## Roles & Permissions
- **Admin**: Full system access.
- **Editor**: Content management (Articles, Comments, Media).
- **Writer**: Can create drafts but requires approval to publish.
- **Subscriber**: Standard user with read access.

## Personas
Personas are AI personalities used by agents. Manage them in **Settings > Personas**.
- **Create**: Define name, role, and tone.
- **Assign**: Link personas to specific AI Agents.

## AI Journalists
Create "virtual" staff members in **Users > AI Journalists**. These are special user accounts powered by AI agents that can "author" articles.
