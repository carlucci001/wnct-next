# Site Configuration

## General Settings
Go to **Settings > General**.
- **Site Name**: The public name of the publication.
- **Description**: Meta description for SEO.
- **Logo**: Upload site logo.

## Modules
Enable or disable features in **Settings > Modules**.
- **Comments**: Turn commenting on/off.
- **Ads**: Enable/disable ad placements.
- **Weather**: Show/hide weather widget.

## Menus
Configure navigation in **Settings > Menus**.
- **Header**: Main navigation links.
- **Footer**: Links at the bottom of the page.

## Credits & Billing
View usage and manage subscription in **Settings > Billing**.
- **AI Credits**: Monitor usage of AI generation features.
- **Invoices**: View past payments.
