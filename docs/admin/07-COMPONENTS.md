# Components

## Directory
Manage local business listings in **Directory**.
- **Add Business**: Create new listings manually.
- **Import**: Bulk import from CSV or Google Places.

## Advertising
Manage ad units in **Advertising**.
- **Placements**: Header, Sidebar, In-Article.
- **Campaigns**: Schedule ads and track impressions.

## Blog
Separate from main news, the Blog component is for less formal updates.
- Managed similarly to Articles but with different templates.

## Events
Manage the community calendar in **Events**.
- **Add Event**: Date, time, location, description.
- **Recurring**: Set up repeating events.

## Community Features
- **Polls**: Create user polls.
- **Forums**: Manage discussion boards (if enabled).
