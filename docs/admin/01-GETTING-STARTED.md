# Getting Started

## Admin Overview
The Admin Panel is the control center for the WNC Times platform. Access it by navigating to `/admin` after logging in with an admin account.

## Navigation
The sidebar provides access to all major sections:
- **Dashboard**: Overview of site stats.
- **Articles**: Manage news content.
- **Newsletters**: Manage email campaigns.
- **Media**: Upload and manage images.
- **Users**: Manage user accounts and roles.
- **Settings**: Configure site options.

## Roles
- **Admin**: Full access to all features.
- **Editor**: Can publish and edit all content.
- **Journalist**: Can write and edit their own content.
- **Subscriber**: Read-only access to front-end content.

## Quick Start
1. **Create an Article**: Go to Articles > New Article.
2. **Invite a User**: Go to Users > Invite User.
3. **Configure Site**: Go to Settings > General to set site name and description.
