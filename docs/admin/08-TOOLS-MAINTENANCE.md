# Tools & Maintenance

## Maintenance
Go to **Tools > Maintenance**.

### Image Migration
- **Move to Cloud**: Migrate local images to Firebase Storage.
- **Optimize**: Compress images for faster loading.

### Content Cleanup
- **Archive Old**: Automatically archive articles older than X days.
- **Delete Drafts**: Remove abandoned drafts.

### Imports
- **CSV Import**: Import articles or users from other systems.
- **RSS Feed**: Ingest content from external feeds.

### System Health
- **Logs**: View error and activity logs.
- **Cache**: Clear system caches.
